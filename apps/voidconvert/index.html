FAQ

stuff_users_ask · before_install

Does VoidConvert need anything else installed?

Image conversions work standalone — Sharp ships with the installer. Video and audio conversions need FFmpeg on your system PATH. On Windows you can install it with `winget install FFmpeg` or grab a build from ffmpeg.org. The app shows a clear error if FFmpeg is missing.

Is it really free?

Yes. No trial, no subscription, no account, no ads. The Windows installer is a one-and-done download. We don't track you or your files — everything runs locally on your machine.

Where do my files go?

Nowhere. VoidConvert runs entirely on your PC. Files never leave your machine, never touch a server, never get uploaded anywhere. Output lands in the folder you pick (defaults to the source folder).

What formats are supported?

Image: PNG, JPG, JPEG, WebP, AVIF, TIFF, GIF, BMP, ICO, HEIF, HEIC, JXL, JP2, RAW, SVG, PSD (input). Video and audio: every common format via FFmpeg — MP4, MKV, AVI, MOV, WMV, FLV, WebM, MP3, WAV, AAC, OGG, FLAC, M4A, OPUS, and more.

Which Windows versions are supported?

Windows 10 and Windows 11 (64-bit). The app is an Electron desktop application — bundles its own runtime, no separate Node install needed.

Is there a mobile version?

An Android build is in closed alpha testing — see the VoidConvert Play Store listing for tester opt-in. Desktop is the production-ready target right now.
