FAQ

stuff_admins_ask · before_install

Is this really free?

Yes. RustCON Desktop is a free Windows download. No account, no subscription, no in-app purchases, no ads. Built solo and released as a thank-you to the Rust admin community.

How is the desktop version different from the mobile RustCON app?

Same workflow, different surface. Desktop is a native Windows app you install from this page. Mobile RustCON ($3.99/mo or $29.99/yr) is the Android + iOS subscription with cloud sync, push alerts, and multi-device profile sync. Use whichever fits your admin habits — or both.

Does RustCON Desktop need a server-side plugin?

No. RustCON connects over standard WebRcon, the same way the in-game F1 console and the official Facepunch tools do. If your server has RCON enabled, RustCON Desktop works.

Does it work with Oxide (uMod) and Carbon?

Yes — both. The permissions manager has native support for Oxide and Carbon group/user/permission models. Toggle between frameworks in the app.

Which Windows versions are supported?

Windows 10 and Windows 11 (64-bit). The installer is signed and bundles its own runtime — no separate .NET or Node install required. macOS and Linux builds are in active development.

Is this affiliated with Facepunch Studios?

No. RustCON is an independent third-party admin tool that talks to the standard Rust RCON endpoint. Rust® is a trademark of Facepunch Studios.
