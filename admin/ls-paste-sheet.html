🍋 LS Paste Sheet

Fastest path to create your 48 products in Lemon Squeezy. Open this page and https://app.lemonsqueezy.com/products/new side-by-side. Click COPY → alt-tab → paste.

Workflow per product:
  1. Click + NEW PRODUCT button to open LS in a new tab
  2. In LS: fill Name → paste the NAME copy from this card
  3. Upload the .cs FILE (path shown at bottom of card)
  4. Set Price → paste the PRICE (cents) value (LS uses cents)
  5. Paste the SHORT DESC into the description box
  6. Click Publish in LS
  7. Back here: check the ✓ Done box to grey out the card
