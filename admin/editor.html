Sales report — one-time setup
This report calls Paddle's API through a server-side proxy so your secret key never touches the browser. Set up takes ~60 seconds:
  1. In Cloudflare Pages → your misfits-studios-site project → Settings → Environment variables, add two secrets (Production scope):
    • PADDLE_API_KEY = your Paddle server API key (pdl_live_apikey_...)
    • EDITOR_ADMIN_SECRET = any random string you choose, e.g. xyz-2026-random
  2. Redeploy (Pages auto-redeploys when env vars change, or trigger manually)
  3. Paste the same EDITOR_ADMIN_SECRET below and hit Save
