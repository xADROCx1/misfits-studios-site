Step 2 — Use it

  1. Go to https://app.lemonsqueezy.com/products/new in your browser
  2. Click the 🍋 Fill Next LS Product bookmark on your bookmarks bar
  3. A purple panel appears top-right with the current product, price, and file path to upload
  4. Manually drag the .cs file shown into the LS upload zone
  5. Toggle Generate license keys on for paid products
  6. Click Publish in LS
  7. Navigate to /products/new again, click the bookmarklet — it auto-advances to product 2, 3, 4...
  8. Progress saves in localStorage so you can resume anytime

What it fills

  • ✓ Name — exact product name from products.json
  • ✓ Price — in cents (999 for $9.99)
  • ✓ Slug — if the field is empty
  • ✓ Description — short description (works with LS's rich-text editor)
  • ✗ File upload — manual (file path shown in banner)
  • ✗ License keys toggle — manual one-click
  • ✗ Publish button — manual (so you can verify each product before going live)

If it can't find a field

LS sometimes renames their DOM selectors. If the banner shows ✗ name field not found or similar, just fill that one field manually — the rest will still auto-fill.
